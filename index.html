    nodeOCR

    A fast implementation of nodecr for a node.js server. Uses node.js, tesseract-3.01, and nodecr.

    Installation

    Install node.js, and tesseract-3.01. Then, using shell, run:

    $ git clone https://github.com/KevZho/nodeOCR.git
$ cd nodeOCR
$ install.bat

    Usage

    $ run.bat

    This sets up a web server at http://localhost:8888. You can then make POST requests with the request body as a base64 encoded jpeg file. The server will then return the OCR'ed text.
